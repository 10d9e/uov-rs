# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-04-09

### Added

- Initial Rust crate implementing UOV signatures.
- Public SDK API: `Scheme`, `KeyPair`, `SigningKey`, `VerifyingKey`, `Signature`.
- KAT test suite (`tests/kat_test.rs`) and Criterion benchmarks (`benches/uov_bench.rs`).
